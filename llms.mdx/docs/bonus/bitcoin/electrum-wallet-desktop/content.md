[Electrum Wallet](https://electrum.org) is a well-established,
feature-rich Bitcoin wallet for power users with broad hardware
wallet support. Pointing it at your own Electrs or Fulcrum server
keeps address lookups private and removes the dependency on public
Electrum servers.
