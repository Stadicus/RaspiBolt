[I2P](https://geti2p.net/en/), the Invisible Internet Protocol, is
an anonymous, end-to-end encrypted network layer where peers never
see each other's real IP addresses. The guide installs
[i2pd](https://i2pd.readthedocs.io/), a C++ I2P daemon, so Bitcoin
Core can reach peers over I2P in addition to clearnet and Tor.
