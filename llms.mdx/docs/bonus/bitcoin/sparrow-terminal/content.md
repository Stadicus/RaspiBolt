Sparrow Terminal is a stripped-down build of [Sparrow Wallet](https://sparrowwallet.com/)
that runs on systems without a graphical display. It's primarily a
configuration utility for running Sparrow as a server daemon on your
node, with the GUI client connecting from another machine on your
network.
