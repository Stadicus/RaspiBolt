[JAM](https://github.com/joinmarket-webui/jam) is a web interface for
JoinMarket that aims to make CoinJoin accessible to non-expert users.
It ships sensible defaults and a browser-based UI while still
exposing the advanced knobs experienced JoinMarket users rely on.
