# Bitcoin bonus (/docs/bonus/bitcoin)



This section covers Bitcoin tooling that goes beyond the core
RaspiBolt setup. You'll find self-hosted block explorers, desktop
and CLI wallets, alternative Electrum server implementations,
privacy layers like JoinMarket and I2P, and a testnet variant of
the whole stack. Pick the pieces that match how you use your node.

</Callout>
