# Bonus (/docs/bonus)



The bonus section collects everything that isn't part of the core
Bitcoin and Lightning node setup, alternative wallets, additional
dashboards, privacy add-ons, quality-of-life tweaks, and experiments.
These guides are optional: the main path works without any of them.

<Callout type="info" title="Content being migrated">

  the v3 source while we work through the backlog.
</Callout>
