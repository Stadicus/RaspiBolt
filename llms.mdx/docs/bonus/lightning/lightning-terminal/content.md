[Lightning Terminal](https://github.com/lightninglabs/lightning-terminal)
is a web UI from Lightning Labs that helps LND operators manage
channel liquidity through the commercial Loop and Pool services. It
bundles those services in a single interface and is useful if you
want managed submarine swaps or liquidity leases without writing
scripts.
