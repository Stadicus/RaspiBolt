# Lightning bonus (/docs/bonus/lightning)



This section gathers Lightning tools that extend what the core
RaspiBolt gives you: browser dashboards, CLI status scripts,
liquidity and fee automation, the Core Lightning implementation as
an alternative or companion to LND, and wallets that talk to your
node over the local network.

<Callout type="info" title="Content being migrated">
  The bonus pages are still being rewritten for v4. Each page links to
  the v3 source while we work through the backlog.
</Callout>
