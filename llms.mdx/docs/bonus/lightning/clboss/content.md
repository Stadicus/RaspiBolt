[CLBoss](https://github.com/ZmnSCPxj/clboss) is an automated manager
for Core Lightning nodes. It picks useful peers to open channels
with, acquires inbound capacity through Boltz swaps, rebalances
existing channels, and sets competitive forwarding fees, so your
CLN node can run largely on autopilot.
