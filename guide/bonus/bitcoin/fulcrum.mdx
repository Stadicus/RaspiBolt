[Fulcrum](https://github.com/cculianu/Fulcrum) is a fast SPV server
for Bitcoin, Bitcoin Cash, and Litecoin written in C++ by Calin
Culianu. Independent benchmarks rank it as one of the quickest
Electrum server implementations available, which makes it a solid
alternative to Electrs when you want lower query latency.
