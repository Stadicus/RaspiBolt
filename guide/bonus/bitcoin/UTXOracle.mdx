[UTXOracle](https://utxo.live/oracle) is a Python script by Daniel Hinton
and Steve Jeffress that derives a close BTC/USD price estimate directly
from on-chain data, without talking to any exchange or external API.
It works because a predictable share of transaction outputs land on
round-dollar amounts, so the blockchain itself carries enough signal
to reconstruct a price.
