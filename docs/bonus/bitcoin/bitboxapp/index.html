BitBoxApp

Beginner-friendly desktop wallet for the BitBox02 hardware wallet, pointed at your own node.

Migration in progress

This bonus page hasn't been rewritten for RaspiBolt v4 yet. The v3 version still works, see the original guide on GitHub while we port it over.

The BitBoxApp is the beginner-friendly companion app to the BitBox02 hardware wallet from Shift Crypto. By configuring it to connect directly to your RaspiBolt instead of Shift's servers, you verify your own transactions and avoid leaking wallet addresses to a third party.
