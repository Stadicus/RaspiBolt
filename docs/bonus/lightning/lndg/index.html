LNDg

Lightweight web GUI for power users to monitor and automate LND routing-node operations.

Migration in progress

This bonus page hasn't been rewritten for RaspiBolt v4 yet. The v3 version still works, see the original guide on GitHub while we port it over.

LNDg is a lightweight web interface for LND aimed at routing-node operators. It surfaces channel and forwarding data and can automate rebalancing, fee adjustments, and channel opens so you spend less time manually running lncli commands.
