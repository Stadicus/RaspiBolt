Bonus

Optional guides for extra tooling, alternative software, and niche setups that complement the main RaspiBolt.

The bonus section collects everything that isn't part of the core Bitcoin and Lightning node setup, alternative wallets, additional dashboards, privacy add-ons, quality-of-life tweaks, and experiments. These guides are optional: the main path works without any of them.
