    Interaction

    Buttons

    Two variants are enough. Primary for the single main action on a view, ghost for secondary affordances.

    Emphasis

    Callouts

    Fumadocs <Callout> component with four types. Use sparingly, one per page, maybe two. Over-use dilutes signal.

    Informational note

    Context or hint that aids understanding. The most common callout.

    Risk or irreversible action

    Used for SSH key backups, 'test before disabling passwords', anything that bites if you get it wrong.

    Validation step

    Used after a command that confirms success, e.g. 'fail2ban-client status sshd' output looks like this.

    Optional shortcut

    Nice-to-know trick or optimisation that isn't required to complete the chapter.

    Navigation

    Cards

    Used at the bottom of overview pages to route readers to the next set of chapters. Three sizes, compact, standard, feature.

    Standard card

    Title + one-line description. Fits 2-up or 3-up grids.

    With icon

    Adds an icon slot for visual scanning.
